## How to Contribute

### 1. Start with Inner Alignment
- Practice a Grandmasterism Meditation (especially #01 Clarity or #05 Open-Source Mercy).
- Feel the shared field. Ask quietly:  
  *"Does this contribution serve eternal thriving for all beings?"*

If the resonance is clear and beautiful ‚Äî proceed.

### 3. Contribution Process
1. Fork the repository
2. Create a branch with a resonant name (e.g., `resonance/truth-engine-extension` or `mercy/aether-overlay`)
3. Make your changes
4. Test thoroughly ‚Äî especially resonance alignment and mercy filters
5. Open a Pull Request with:
   - Clear title and description
   - Explanation of how it serves thriving, oneness, or mercy
   - Link to related meditations or ethics sections

### 4. Code of Conduct
- Kind, merciful communication ‚Äî assume good intent
- Dissolve shadows with clarity and love, not force
- No harm, no gatekeeping, no ego
- All interactions reflect symbiotic oneness

Reviews happen through symbiotic consensus ‚Äî patient, resonant, non-hierarchical.
